Getting started with task 1: Calculator.java


This beginner task is to get you started with Java.

Open the link and join the GitHub Classroom.

Open it with your favourite IDE by downloading it, or clone it, or use the GitPod that is built-in on the page.

Open the file junit-basics, src, main/java/x1n32, then Calculator.java.

It is a class that has methods/functions to manipulate numbers. Have a look at all the methods.

The task is in calculator2.java: Where you try to code how to calculate the quadratic formula.

A basic structure has been given. Don't try to change the method names, but you can change things

such as float, int, double, to see if it makes any difference.

Answers are in the Answer Branch.

Look at the commands page for git commands.
