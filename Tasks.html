This page provides some coding tasks to guide you through certain topics. As a challenge, try to ask a friend to join you on these tasks.

Getting started with task 1: Calculator.java


This beginner task is to get you started with Java.

Open the link and join the GitHub Classroom.

Open it with your favourite IDE by downloading it, or clone it, or use the GitPod that is built-in on the page.

Open the file junit-basics, src, main/java/x1n32, then Calculator.java.

It is a class that has methods/functions to manipulate numbers. Have a look at all the methods.

The task is in calculator2.java: Where you try to code how to calculate the quadratic formula.

A basic structure has been given. Don't try to change the method names, but you can change things

such as float, int, double, to see if it makes any difference.

Answers are in the Answer Branch.

Look at the commands page for git commands.


Task 2: Data Structures


This task builds upon the use of data structures in Java, such as Arrays, lists, and maps.

The classroom link will provide you some code for you to look at.

This task is more broken up than the first, in that is a starter task to review key terms and predict code in the worksheet.txt

Demo code has been provided to show how arrays and lists (linked and array) are made.

After commenting and running it, your task is to extend/write/fix code in HashMapTask and Comic Book that uses HashMap

As a challenge, make a new branch to write your code, and merge it to the main one when you think you are done.
