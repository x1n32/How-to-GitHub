What is on this webpage?

A General guide on GitHub and git commands.

Links to some java code to practice the basics, and learn about Unit Testing with JUnit


Aims and Objectives

To create a usable and understandable skills-aid demo for Keele Computer Science students to showcase the collaboration features of Collaborative Development Tools/ Version Control Systems that could be used for all programming and design modules.

How to get there:

  1. Research what students may need to learn that are relevant to their studies
  2. Research relevant studies that could help improve the quality and usability of the tutorials
  3. Create an information/skills page using GitHub (Pages/Classroom).
  4. Create two-to-three meaningful, but easy-to-implement Use Case scenarios (increasing in difficulty from beginner level) demos showcasing a specific topic in programming, where errors are purposely produced to showcase Unit Testing and Continuous Integration, so that students can learn from mistakes via meaningful error messages and so they can learn by doing.
  5. Create a questionnaire/survey to examine the quality, usability, and ethics of the project.
