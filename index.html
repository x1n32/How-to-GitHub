Welcome to the home page!

Purpose:

To create a usable and understandable skills-aid demo for Computer Science students to improve knowledge of Collaborative Development Tools/ Version Control Systems.

Here you can practice your coding skills via tasks that get marked. You can copy the code via links to GitHub Classroom, or via forking the IDE code on Replit.com


Below is an example of a task you can practice. On the IDE click on the Show Files tab and go to src/main/java/github/x1n32/DemoTest.jav. Click the green run button. When it finishes running, it will report the test results to check if what you programmed is correct!

Have a go at Task 1 when you are ready: Calculator.java
